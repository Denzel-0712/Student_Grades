# Student Grade Calculator in QBasic

## Overview
This is a simple student grade calculator program written in QBasic. It allows users to input student scores, calculates the total and average, and assigns a grade based on predefined grading criteria.

## Features
- Accepts user input for multiple subject scores.
- Calculates the total and average score.
- Assigns a grade based on the average score.
- Provides a user-friendly interface.

## Requirements
- QBasic Interpreter
- A system that can run QBasic (Windows with QB64 recommended)

## How to Run
1. Open QBasic or QB64.
2. Load the `grade_calculator.bas` file.
3. Run the program by pressing `F5`.
4. Enter the number of subjects and their respective scores.
5. The program will display the total, average, and final grade.

## Grading Criteria
The grading is assigned based on the following scale:
- Average ≥ 90: A
- Average ≥ 80: B
- Average ≥ 70: C
- Average ≥ 60: D
- Average < 60: F

## Example Usage
```
Enter the number of subjects: 3
Enter score for subject 1: 85
Enter score for subject 2: 90
Enter score for subject 3: 78

Total Score: 253
Average Score: 84.3
Grade: B
```

## Customization
- Modify the grading scale in the QBasic code to match your grading system.
- Adjust the program to accept additional inputs such as student name or subject names.
